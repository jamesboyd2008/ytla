
To visualize the data:

1. SSH or screen share onto the machine on the mauna
2. cd anaconda3/
3. conda install -c conda-forge plotly
4. (skip this step) pip install pymongo
4. pip install mongoengine
5. conda install -c anaconda git
6. git clone https://github.com/jamesboyd2008/[NAME_OF_REPOSITORY].git
7. conda install -c anaconda virtualenv (maybe unnecessary)
8. pip install plotly (if command #3 didn't already work)
9. pip install msgpack
10. pip install requests
11. conda install -c enversion httpd
12. pip install mod_wsgi
13. cd [NAME_OF_REPOSITORY]
14. mod_wsgi-express start-server mod_wsgi_interface.py &
15. navigate to localhost:8000 in your favorite web browser
